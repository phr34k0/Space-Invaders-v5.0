Installation
-----------

```
Download the whole zip and open main scene in /Assets/Game Assets/_Scenes with your Unity3D

Or

Just download "Phr34kz Space Invaders_Data" folder with "Phr34kz Space Invaders.exe" to play on your winblows
```

Usage
-----

Controls are

* Spacebar = Shoot
* A = Move Left
* D = Move Right
